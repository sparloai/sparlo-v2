# Söhne Fonts

This directory is for the Söhne font files (licensed).

## Required Files

Place the following font files in this directory:

- `Soehne-Buch.woff2` (Regular, 400)
- `Soehne-Kraftig.woff2` (Medium, 500)
- `Soehne-Halbfett.woff2` (Semibold, 600)

Optional for monospace:
- `SoehneMono-Buch.woff2` (Mono Regular, 400)

## Font Fallback

If fonts are not present, the system will fall back to Inter (loaded from Google Fonts).
