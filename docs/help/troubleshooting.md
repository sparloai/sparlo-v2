# Troubleshooting

Solutions to common issues you might encounter.

## Report Issues

### Report is taking too long

If your report seems stuck:
1. Check the progress indicator - complex topics take longer
2. Wait up to 15 minutes for Deep Dive reports
3. If stuck beyond 15 minutes, try refreshing the page
4. If the issue persists, create a new report

### Report generation failed

If you see an error:
1. Check your internet connection
2. Try generating the report again
3. Use a more specific topic if the query was too broad
4. Contact support if the error continues

### Report content seems incomplete

If sections are missing:
1. The topic may not have enough available information
2. Try a more specific query
3. Use a Deep Dive report for more comprehensive analysis

## Sharing Issues

### Share link not working

If recipients cannot access your shared report:
1. Verify the link was copied correctly
2. Check that the report is still available (not deleted)
3. Generate a new share link
4. Contact support if the issue persists

### PDF export failed

If PDF generation fails:
1. Wait a moment and try again
2. Check your internet connection
3. For long reports, PDF generation may take a few minutes
4. Try a different browser if the issue continues

## Account Issues

### Cannot log in

If you cannot access your account:
1. Verify your email address is correct
2. Click "Forgot password" to reset
3. Check your spam folder for the reset email
4. Contact support if you still cannot access your account

### Email not verified

If prompted to verify your email:
1. Check your inbox for the verification email
2. Check spam/junk folders
3. Click "Resend verification email"
4. Wait a few minutes and check again

### Subscription not active

If features are limited unexpectedly:
1. Go to Settings > Billing
2. Check your subscription status
3. Verify payment method is valid
4. Contact support for billing issues

## Performance Issues

### Pages loading slowly

If the app feels slow:
1. Check your internet connection
2. Clear your browser cache
3. Try a different browser
4. Disable browser extensions temporarily

### Features not working

If buttons or features don't respond:
1. Refresh the page
2. Clear browser cache and cookies
3. Try a different browser
4. Check if the issue is specific to one feature

## Getting More Help

If you cannot resolve your issue:
1. Use the chat feature on this page
2. Describe your issue in detail
3. Include any error messages you see
4. Our team will respond as soon as possible
